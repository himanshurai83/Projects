# Collection of Web Development Projects

This repository contains a collection of full-stack and frontend projects built using MERN Stack, JavaScript, Tailwind CSS, HTML, and CSS. These projects demonstrate skills in backend development, responsive UI design, and clean component-based development.

Projects Included

# 1. E-Commerce Website (MERN + Tailwind CSS)

A fully functional e-commerce application built using the MERN stack and styled with Tailwind CSS.
Features:

User authentication (login/signup)

Product listing & detailed product pages

Cart management & checkout

Tailwind CSS responsive UI

Admin panel for product and user management

# 2. Todo App (MERN + Tailwind CSS)

A task-management app with full CRUD functionality and a modern UI created using Tailwind CSS.
Features:

Add, update, delete, and complete tasks

Clean and minimal Tailwind UI

Backend API using Node.js + MongoDB

# 3. Currency Converter (JS + Tailwind CSS)

A web-based currency converter styled with Tailwind CSS and powered by real-time exchange rate APIs.
Features:

Live currency conversion

Dropdown selection for multiple currencies

Simple, clean, responsive interface

# 4. Digital Clock (HTML + CSS + JS)

A real-time digital clock displaying hours, minutes, and seconds.
Features:

Live time updates

Minimal and clean UI

# 5. Calculator (HTML + CSS + JS + Tailwind CSS option)

A simple calculator supporting basic arithmetic operations.
Features:

Responsive layout

Clean button UI (Tailwind or custom CSS)

# 6. Mini Games (HTML/CSS/JS)

A collection of browser-based games designed with simple and responsive layouts.
Examples include:

Rock Paper Scissors

Tic Tac Toe

Number Guessing Game

Snake (if included)

# Tech Stack

Frontend: HTML, CSS, Tailwind CSS, JavaScript, React

Backend: Node.js, Express

Database: MongoDB

Tools: Git, GitHub, REST APIs
